# How to Find Links

To find working Ghost links, join the official community server and complete verification first.

## Recommended Process

- Join the [Ghost Discord server](https://discord.gg/UZzYt4uE6D).
- Complete the verification process.
- Check official link channels first.
- Review trusted shared links from the community.
- If needed, deploy your own link for full control.

## Best Practices

- Prefer official or self-hosted links for reliability.
- Avoid reposted links from unknown sources.
- Keep backup links in case an instance is offline.
