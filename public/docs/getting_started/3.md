# General Information

Ghost is a browser-style web client focused on accessibility, customization, and user-side control.

## Privacy and Data

- Ghost stores user data locally by default.
- You can remove all local data from Settings at any time.
- You can export and import data using Ghost backup files.

## Reliability

- Availability depends on your selected backend, transport, and network conditions.
- Some websites may fail due to upstream restrictions, certificate issues, or temporary server downtime.
