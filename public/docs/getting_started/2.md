# How to Use Ghost

Ghost functions similarly to a standard web browser, with a workflow and layout inspired by modern tab-focused browsers.

## Core Features

- Multi-tab browsing with quick switching and tab persistence.
- Search and URL navigation through the omnibox.
- Built-in app and media sections for games, TV, music, AI, and tools.
- Configurable proxy backends and transport options.
- Integrated privacy and anti-detection controls.

## Basic Workflow

- Open or create a tab.
- Enter a URL or search query in the omnibox.
- Use sidebar shortcuts to navigate Ghost sections quickly.
- Use Settings to customize theme, behavior, and performance.

## Browser Controls

- Back, forward, and reload are available in the top bar.
- Bookmarks can be added from the top bar.
- Per-site controls (ad block, popup block, download block) are available where applicable.

## Tips for New Users

- Configure your preferred search engine and backend in Settings.
- Enable Performance Mode if your device is resource-constrained.
- Export your data regularly to keep local backups.
