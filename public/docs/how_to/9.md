# Access My Home Computer with WarpDesk

WarpDesk is Ghost's beta remote desktop experience. It is usable now, but still being polished.

## What You Need

- A WarpDesk host running on the machine you want to reach.
- The host's Connection URL.
- Your WarpDesk login credentials.

## How to Connect

1. Open Ghost and launch WarpDesk from the Remote Access page.
2. Paste the host Connection URL into the login form.
3. Enter your username and password.
4. Press Connect to start the session.

## Beta Note

- WarpDesk is in beta, so occasional bugs or compatibility issues are expected.
- If you are connecting from outside your local network, use a tunnel or HTTPS endpoint.
