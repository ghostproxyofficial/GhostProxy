# Speed Up Searching

If search feels slow, optimize your routing and runtime settings.

## Recommended Steps

1. Host and use your own Wisp server.
2. Enable Performance Mode.
3. Switch backend to Scramjet where appropriate.
4. Improve network quality (Wi-Fi stability and latency).

## Additional Guidance

- Test one change at a time and measure response.
- Use reliable endpoints and avoid overloaded public relays.
