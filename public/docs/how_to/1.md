# Customize

Ghost provides extensive customization controls for appearance and browser behavior.

## What You Can Customize

- Theme selection, including preset themes.
- Custom theme creation.
- Typography (global font selection).
- Navigation and UI scaling options.
- Performance-related visual toggles.

## How to Customize

- Open Settings.
- Go to Customize.
- Select your preferred theme and font.
- Adjust options in other settings categories to match your browsing workflow.

## For Self-Hosted Instances

- You can apply custom global CSS to modify visual behavior and styling for your deployment.
