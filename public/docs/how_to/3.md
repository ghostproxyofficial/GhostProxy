# Import/Export My Data

Ghost allows local backup and restore through export and import tools.

## Export Data

- Open Settings.
- Go to Data.
- Select Export Data.
- Save the generated `.ghost` backup file.

## Import Data

- Open Settings.
- Go to Data.
- Select Import Data.
- Choose a previously exported `.ghost` file.

## Additional Backup Option

- In the Profile Manager, you can also export profile-related data for later restore.

All exported/imported data is local to your environment.
