# Fix Common Issues

Use this guide to identify common proxy and transport errors in Ghost.

## LibCurl Errors

- SSL connect error 7: The Wisp server is currently unavailable.
- SSL connect error 35: The destination website may not exist or is unreachable.
- SSL connect error 60: The destination website certificate is malformed, expired, or invalid.

## Epoxy Errors

- WebSocket failed to open: The Wisp server is currently unavailable.
- Stream creation failed due to an unreachable destination: The destination website may not exist or is unreachable.

## Scramjet and BareMux Errors

- There are no bare clients: Request was issued before workers were fully registered. Refresh the page and try again.
- Attempted to fetch from same origin: Scramjet attempted to fetch the host instance URL directly. Some sites with unusual refresh behavior can trigger this. Use Back and retry navigation.

## General Recovery Steps

- Refresh the tab.
- Switch transport/backends in Settings.
- Verify your selected Wisp server is online.
- Confirm the destination website is currently reachable from your network.
