# How to Host/Deploy a Client

Deploying a client means creating and hosting your own Ghost instance.

## Standard Workflow

1. Open the Ghost repository: [ghostproxyofficial/GhostProxy](https://github.com/ghostproxyofficial/GhostProxy).
2. Follow `README.md` instructions to clone the project locally.
3. Apply any code or branding changes you want (optional).
4. Push your version to a new Git repository.
5. Deploy using Cloudflare Pages (recommended) or GitHub Pages.

## Recommendation

- Cloudflare Pages is preferred for simplicity, speed, and deployment reliability.
