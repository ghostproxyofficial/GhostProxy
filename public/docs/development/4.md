# How to setup WarpDesk (Ghost Remote Access)

WarpDesk is a remote proxy protocal brought to you by the developers of Ghost, to access your home computer from outside.

## Standard Workflow

1. Open the Ghost repository: [ghostproxyofficial/WarpDesk](https://github.com/ghostproxyofficial/WarpDesk).
2. Follow `README.md` instructions to clone the project locally.
3. Apply any code or changes you want to fit your needs.
4. Follow `README.md` to configure your local build
5. Run the app, and forward/mirror the server port.

## Server Port Handeling

- You can setup your own TURN server for connection, but we recommend you use Cloudflare
- Cloudflare provides Tunneling (exposing your outbound-only server port)
- Cloudflare provides a TURN server (maintaining the connection)
- These features are avaliable for free, read `README.md` for real steps
