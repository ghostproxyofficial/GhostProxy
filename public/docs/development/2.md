# How to Set Up Remote Proxy Server

This is an advanced feature for users who want traffic routed through an additional proxy layer.

## Requirements

- A running proxy endpoint.
- Supported protocol: HTTP, SOCKS4, or SOCKS5.

## Configuration Steps

1. Open Settings (or Quick Browsing Settings).
2. Set Proxy Routing to Remote Proxy Server.
3. Select proxy type: HTTP, SOCKS4, or SOCKS5.
4. Enter your proxy endpoint.

## Notes

- Incorrect protocol/endpoint combinations will cause connection failures.
- Validate your proxy endpoint outside Ghost first when troubleshooting.
